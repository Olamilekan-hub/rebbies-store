# Rebbie's Store
## Nigeria's Premier Hair & Jewelry E-commerce Platform

### Tech Stack
- **Backend**: MedusaJS (Node.js)
- **Frontend**: Next.js with TypeScript
- **Database**: PostgreSQL
- **Payments**: Paystack & Flutterwave
- **Hosting**: Render + Vercel

### Getting Started

#### Backend Setup
```bash
cd backend
npm install
npm run dev
```

#### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
